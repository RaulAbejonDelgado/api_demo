API REST DOCUMENTACIÓN

Api para la práctica JavaEE Rest Full (hatoas) con MongoDB, usando el ODM Morphia y spring batch

La siguiente api nos ofrece las operaciones crud básicas de los objetos : personas, familias, comentarios

Index del API http://localhost:8080/API

Código fuente de la aplicación servidor https://github.com/RaulAbejonDelgado/api_demo

Código fuente de la aplicación cliente https://github.com/RaulAbejonDelgado/cliente_api_demo

Recurso GET GET/{id} POST PUT/{id} DELETE/{id}
http://localhost:8080/API/personas Listado de personas Detalle de persona Requisitos:
  • Nombre: Requerido
  • Nombre: Texto
  • Nombre: Entre 5 y 150 caracteres
  • Familia: Requerido
  • Familia: Valor numerico entero
Respuestas:
  • 201 Creado
  • 409 incumplimientos de alguna de los requisitos
objeto esperado:

{
    "nombre": "enhord",
    "familyId":1
}
